# 算数パズルバトル

数字パネルをなぞって合計10を作り、敵を倒すパズルRPGです。

## ゲーム概要

画面に並んだ数字パネルを縦・横になぞり、合計がぴったり **10** になるように組み合わせて敵にダメージを与えます。
たくさんのパネルをつなげるほど、そしてコンボを重ねるほど威力が増していきます。

---

## モード紹介

### ⚔️ ダンジョンモード
5段階のレベルを攻略するRPGモードです。
15秒の操作タイムで敵にダメージを与え、敵の反撃を凌ぎながらHPをゼロにすれば勝利です。

| LEVEL | 敵 |
|---|---|
| 1 | スライム |
| 2 | ゴブリン |
| 3 | サラマンダー |
| 4 | アルティメットスライム |
| 5 | ドラキュラ |

### 📚 トレーニングモード
99秒のタイムアタックモードです。制限時間内にどれだけ高いスコアを出せるか挑戦します。
消えたパネルの下に隠し絵が出現するおまけ要素もあります。

---

## 操作方法

- **縦・横スライド** → パネルをつなげて合計10を作る
- **斜めドラッグ** → 矩形範囲でまとめて選択
- **途中まで戻る** → 選択ラインを短くする
- **消えたマスを越える** → 空白を飛び越えてつなげられる

---

## ダメージ・スコア計算

| 条件 | ダメージ / スコア |
|---|---|
| 2〜3個で10 | 200 |
| 4個で10 | 250 |
| 5個で10 | 300 |
| N個で10（4個以上） | 200 + (N-3) × 50 |
| ぴったり20 | +800（バースト回復も発動） |
| ぴったり30（トレーニング限定） | +1500 + 盤面リセット |

**コンボボーナス：** 同ターン中に繰り返し10を作るたびに倍率が上がります
（1回目 ×1.0 → 2回目 ×1.1 → 3回目 ×1.2 …）

## キャラクター・バースト

Attackボタン表示中にキャラアイコンをタップして発動。各キャラ1回のみ使用可能です。

| キャラ | 必殺技 | 効果 |
|---|---|---|
| 👧 勇者 | ゴーゴードライブ | 攻撃後の盤面でパネルの約30%を5に変える |
| 🐉 コドモドラゴン | ドラゴンソウル | そのターンの攻撃を2回行う |
| 🦄 ユニコーン | ホーリーホーン | 与えたダメージの5倍分HPを回復する |

> ぴったり20達成でドラゴン・ユニコーンのバーストが復活します（勇者は除く）
